# Orion Soultion Group


### Prerequisites

- [DynamoDB]
- [Yarn] (https://yarnpkg.com/lang/en/docs/install/)
- [Node.js and npm](nodejs.org) Node >= 4.x.x, npm >= 2.x.x
- [Homebrew] (https://brew.sh/)
- [Angular CLI]



## Compile and run Project
1. "clone repo"
2. "npm install"
3. "yarn start"

## Start Node Server
1. cd into orion/server
2. type "node server.js"
